All Your Files Kept Minimal.
Run `ayfkm` after `npm install` to trim out cruft.

Put this in your package.json `devDependencies`:

    "ayfkm": "*"

then run this:

    npm install && ./node_modules/.bin/ayfkm

# Install

    npm install ayfkm
